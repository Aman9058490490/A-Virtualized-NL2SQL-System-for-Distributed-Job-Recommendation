# Screenshots and Visual Assets

Add screenshots of your application here to showcase the modern UI.

## Recommended Screenshots

1. **Home Page / Query Input**
   - Shows the main query input interface
   - Example queries visible
   - Dark/Light mode toggle

2. **SQL Display**
   - Syntax-highlighted SQL queries
   - Tabbed interface (Course DB / Job DB)
   - Copy-to-clipboard buttons

3. **Results Table**
   - Data table with results
   - Export buttons (CSV/JSON)
   - Tab navigation (Merged/Course/Job)

4. **Final Answer**
   - AI-generated natural language answer
   - Gradient styling
   - Complete end-to-end result

5. **Dark Mode**
   - Application in dark theme
   - Show modern glassmorphism effects

## How to Add Screenshots

1. Take screenshots of the running application
2. Save as PNG files in this directory
3. Name them descriptively (e.g., `home-light.png`, `results-dark.png`)
4. Reference them in README.md

## File Naming Convention

```
screenshots/
├── 01-home-light.png
├── 02-home-dark.png
├── 03-sql-display.png
├── 04-results-table.png
├── 05-final-answer.png
└── 06-mobile-responsive.png
```
